# RidePilot AI Website

## Deploy to GitHub Pages

1. Create a new repo at github.com named `ridepilot-site`
2. Upload these files:
   - index.html
   - privacy.html
   - README.md

3. Go to repo Settings → Pages → Source: Deploy from branch → main → / (root)
4. Your site will be live at: https://yourusername.github.io/ridepilot-site

## Custom Domain (optional)
- Buy ridepilotai.app at namecheap.com (~$12/year)
- Add CNAME file with: ridepilotai.app
- Point DNS to GitHub Pages

## Apple App Store requires:
- Privacy Policy URL: https://yourusername.github.io/ridepilot-site/privacy.html
- Support URL: mailto:support@ridepilotai.app
- Marketing URL: https://yourusername.github.io/ridepilot-site
